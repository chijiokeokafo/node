Web Development

I have experience with the majority of the modern web development toolbelt, including HTML5, CSS, JavaScript, jQuery, Wordpress, SilverStripe, Ruby on Rails, Bootstrap and Git/Github. I can build pixel perfect PSD to HTML chops, custom (SilverStripe) or Wordpress blogs/portfolio sites, web apps and banner ads.

HTML5 Banner Development

HTML5 banner ads have been one of my main development focuses since beginning my career. Using programs like Google Web Developer and Adobe Edge Animate, and raw HTML/JS/CSS, I've created pixel perfect, IAB compliant banners for clients such as Purina, BMO, Vanguard, Porsche and Subaru.

Work Ethic

Whether it's designing and building a new custom bike for myself, trying to land a trick, or building a website for someone. I stay focused and I work hard until the job is done.
